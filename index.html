The jMoka Engine

jMoka Engine is an Open Source 2D Game Engine written in Java. Built with a (slightly simple) entity-component system that will be exposed in the documentation pages.

Currently, the engine is still in an early development stage, but already has some nice ideas that are worth of documentation, hence this web pages. Here you can see a list of some of them:

  • Prefabs described in text files written in Oping, a simple descriptive language built for the engine.
  • Multiple scenes with cycle states and callbacks.
  • Describe your resources in one place and access them inside text files, the engine will take care of the linking.
  • Resources can be anything (every Object is allowed).
  • A strong Java API that relies on commonly used design patterns.
  • Multiple cameras.
  • Fully supported by OpenGL and OpenAL*.
  • Fully extensible.
  • Component packages, so you can reuse them in a lot of projects.
  • Components are the base mechanism of the engine, you define them in the prefabs files.
  • Components can be created to work with a fully customizable event system. Every event if driven by Triggers.
